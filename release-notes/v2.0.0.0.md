# Enhanced Vehicle Squared 2.0.0.0

This is the first release under the independent Enhanced Vehicle Squared name.
The original project was closed on 17 July 2026 after about 275 days without a
substantive code release. Its owner added an explicit development-stopped
notice, archived the repository as read-only, closed both outstanding pull
requests unreviewed and unmerged, and closed all three outstanding issues as
not planned. No maintainer response accompanied those closures, and the public
record identifies no successor or transition plan. The prolonged release
inactivity, explicit cessation, unanswered work, and archival left no upstream
maintenance path, so Squared now independently maintains, reviews, tests, and
releases this continuation.

## Changes

- Rebranded the maintained project, in-game title, icon, documentation, and
  release line as Enhanced Vehicle Squared.
- Corrected guidance offset wrapping when crossing track boundaries.
- Corrected grouped front/rear folding using Farming Simulator fold-state and
  fold-direction contracts.
- Corrected grass and headland classification using the installed-game ground
  type mapping.
- Made current configuration files authoritative over legacy files and added
  verified, deterministic migration behavior.
- Preserved malformed configuration files for manual recovery.
- Prevented dedicated-server configuration file access and isolated client-only
  HUD, GUI, renderer, and sound resources.
- Hardened licensed-test cleanup, protected mod settings, structured markers,
  archive comparison, and validated-artifact publication.
- Changed distributed text to English only.

## Compatibility

The download intentionally remains `FS25_EnhancedVehicle.zip`. Replace the old
ZIP in place and do not rename it or install both versions. The legacy filename,
input actions, savegame keys, Lua identifiers, and configuration path are kept
so existing saves, custom bindings, and settings continue to work.

Requires Farming Simulator 25 version 1.20.0.0 or newer.

## Validation scope

The release candidate passed Lua 5.1 syntax checks, proprietary-free runtime
and configuration tests, installed-game source-contract checks, deterministic
archive checks, licensed local client acceptance, licensed local dedicated-
server acceptance, and protected-state restoration checks.

Networking behavior has automated simulated coverage. Genuine multi-machine
multiplayer was not tested, and subjective visuals still require human review.

See `ATTRIBUTION.md` inside the ZIP for original-work credit and project lineage.
