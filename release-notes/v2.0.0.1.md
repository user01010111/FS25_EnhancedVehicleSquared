# Enhanced Vehicle Squared 2.0.0.1

This release completes the source-cleanup and current-maintenance transition
for the independent Enhanced Vehicle Squared project. The former project was
explicitly discontinued and archived as read-only on 17 July 2026 after a long
period without a substantive release or a public maintenance transition.

## Changes

- Removed four unreferenced legacy working assets that were never shipped in
  the production mod.
- Removed the embedded historical changelog, decorative separators,
  commented-out debugging, redundant UI section comments, and self-evident
  source commentary.
- Retained concise English documentation for engine quirks, compatibility
  interfaces, network authority, configuration migration, cleanup ordering,
  renderer contracts, and non-obvious mathematics.
- Identified `user01010111` as the current maintainer and release coordinator with
  Enhanced Vehicle Squared contributors.
- Centralized original-work and historical-contributor credit in
  `ATTRIBUTION.md` while preserving the complete license and non-endorsement
  notice.
- Added automated enforcement for English-only translations, attribution
  boundaries, removed legacy assets, source hygiene, and legal release files.

## Compatibility

Runtime behavior and the compatibility ABI are unchanged from 2.0.0.0. The
download remains `FS25_EnhancedVehicle.zip`; input actions, savegame keys,
configuration paths and migration, network identity, specialization keys, and
public Lua interfaces remain unchanged.

Requires Farming Simulator 25 version 1.20.0.0 or newer.

## Validation scope

The exact release source passed Lua 5.1 syntax and behavior checks,
configuration and client-runtime tests, installed-game source-contract checks,
deterministic packaging and source/archive equality checks, licensed local
client acceptance, licensed local dedicated-server acceptance, and protected
state restoration checks. Production Lua bytecode and the parsed UI XML tree
were also compared with 2.0.0.0 to verify that the cleanup changed no runtime
behavior or UI structure.

Automated networking tests simulate protocol behavior. Genuine multi-machine
multiplayer was not tested, and subjective visuals still require human review.

See `ATTRIBUTION.md` inside the ZIP for original-work credit, modification
notices, contributor history, and project lineage.
